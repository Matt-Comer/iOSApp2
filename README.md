### Description

This SwiftUI Scavenger Hunt app allows users to explore local businesses, follow clues, and find hidden items throughout the community.

The app includes a custom animated splash screen, navigation between views, hidden item tracking, and a rewards system for participants.

### Features

* Animated Splash Screen
* NavigationStack Navigation
* Scavenger Hunt Item List
* Item Detail Screens
* Reward Tracking
* Submit Results Screen
* SwiftUI Interface
* Custom Assets and Images
