# iOSApp2 - Scavenger Hunt App

## Assignment 3

This SwiftUI app is a Scavenger Hunt app for local businesses. Users can view hidden items, read clues, and tap each item to see more details.

## Features

- 10 scavenger hunt items
- Local business names
- Clues for each item
- Image assets for every item
- List screen
- Detail screen
- Navigation between screens

## Technologies Used

- Swift
- SwiftUI
- Xcode
- GitHub

## App Structure

- `ScavengerItem.swift` stores one scavenger hunt item.
- `ScavengerData.swift` stores the scavenger hunt data.
- `ScavengerListView.swift` displays all items.
- `ScavengerDetailView.swift` displays item details.
- `Assets.xcassets` stores all item images.
